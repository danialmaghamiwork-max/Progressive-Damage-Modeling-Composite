# Progressive-Damage-Modeling-Composite
MATLAB code for progressive damage modeling in composite laminates using CLT

توسعه کد محاسباتی (در محیط متلب) بر اساس تئوری کلاسیک لایه‌ها (CLT) جهت تحلیل تنش، کرنش و رفتار مکانیکی مواد مرکب.

مدلسازی خرابی پیش‌رونده (Progressive Damage) برای یک ورق کامپوزیتی شبه‌ایزوتروپیک (T300/5208) با چیدمان ۲۴ لایه.

محاسبه و پیش‌بینی بار شروع خرابی (FPF - First Ply Failure) و بار خرابی نهایی (LPF - Last Ply Failure) تحت بارگذاری افزایشی.

پیاده‌سازی و مقایسه معیارهای مختلف شکست کامپوزیت‌ها شامل: حداکثر تنش (Maximum Stress)، تسای-وو (Tsai-Wu)، هاشین (Hashin) و هاشین اصلاح‌شده.

تحلیل و استخراج ترتیب خرابی لایه‌ها (Failure Sequence) بر اساس معیارهای شکست مختلف.
